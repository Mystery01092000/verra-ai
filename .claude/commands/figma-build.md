description: Build/update the Verra design in Figma from the prototype (requires Figma Dev Mode MCP).

Use the `verra-figma-build` skill (and load the Figma plugin skills first). Follow the build order:
variables → core components → marketing frames → app frames → variants/states → handoff. Bind colors
and type to Figma variables; use auto-layout. See `.claude/connectors/README.md` to connect Figma.
